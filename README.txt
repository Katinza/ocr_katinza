Οδηγίες σωστής λειτουργίας του κώδικα:
Το πρόγραμμα λειτουργεί στην έκδοση 3.10 της python


Απαραίτητη είναι η εγκατάσταση ενός interpreter (π.χ. του Visual Studio Code )
To επίσημο site για την εγκατάσταση του VS Code:  https://code.visualstudio.com/


Εγκατάσταση βιβλιοθηκών


Στο terminal γράφουμε: pip install -r requirements.txt  ώστε να εγκαταστήσουμε τις απαραίτητες βιβλιοθήκες.
 
Για την εγκατάσταση της pytesseract:
Ακολουθούμε τις οδηγίες των site:
https://stackoverflow.com/questions/46140485/tesseract-installation-in-windows
https://codetoprosper.com/tesseract-ocr-for-windows


Αναλυτικές οδηγίες για την εγκατάσταση του pytesseract βήμα προς βήμα εδώ:


Βήμα 1ο 
Κατεβάζουμε το αρχείο tesseract από αυτό το link https://digi.bib.uni-mannheim.de/tesseract/tesseract-ocr-w64-setup-v5.0.0-alpha.20210506.exe .
Βήμα 2ο 
Μετά την εγκατάστασή του ανοίγουμε το αρχείο αυτό. Επιλέγουμε γλώσσα(Αγγλικα) -> συμφωνούμε με τους όρους -> πατάμε next -> τσεκάρουμε την επιλογή install for everyone using this computer -> αντιγράφουμε την τοποθεσία λήψης του αρχείου -> πατάμε διαδοχικά δύο φορές next και τέλος install.
Βήμα 3ο
Στη μπάρα αναζήτησης του μενου εναρξης του υπολογιστή μας πληκτρολογουμε ‘Επεξεργασία των μεταβλητών περιβάλλοντος του συστήματος’ ή εναλλακτικά ‘Edit the system environment variables’.-> Στο παράθυρο που εμφανίζεται πατάμε στο ‘Επεξεργασία μεταβλητων περιβάλλοντος’ -> Επιλέγουμε στο δευτερο πλαίσιο το Path και πατάμε ‘επεξεργασία’. -> πατάμε δημιουργία και κάνουμε επικόλληση την τοποθεσία λήψης που είχαμε αντιγράψει και πατάμε οκ.




Αφού ολοκληρωθεί η εγκατάσταση το πρόγραμμα είναι έτοιμο για να 'τρέξει'. Πατάμε "run" στο αρχείο optical_character_recognition.py ή εναλλακτικά, αφού σιγουρευτούμε πως το terminal βρίσκεται η διεύθυνση του φακέλου στον οποίο βρίσκεται το αρχείο πληκτρολογούμε python optical_character_recognition.py. 


Για το ocr με το pytesseract επιλέγουμε το πάνω δεξιά κουμπί, επιλέγουμε εικόνα και πληκτρολογούμε το όνομα αρχείου. Το αποτέλεσμα θα εμφανιστεί αυτόματα στην οθόνη αλλά και σε αρχείο txt. Για το ocr με TensorFlow επιλέγουμε το κατω αριστερά κουμπί και ύστερα την εικόνα. Έπειτα επιλέγουμε το κάτω δεξιά κουμπί για την επεξεργασία της εικόνας. Το αποτέλεσμα θα εμφανιστεί αυτόματα αλλά και σε αρχείο text αφού μας ζητηθεί η εισαγωγή ονόματός του.
